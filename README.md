Задание
Разработать алгоритм, составить и отладить программы на языке Java по обработке файла данных "Список "толстых" журналов", в котором указаны номера по порядку, наименование журнала, ФИО главного редактора, стоимость одного номера.
Сформировать файлы:
1.	Список главных редакторов журналов с указанием наименований соответствующих журналов;
2.	Список журналов, стоимость которых > min и < max, значения min и max вводятся с клавиатуры;
3.	Определить самый "дешевый ", самый "дорогой " журнал и среднюю стоимость журнала.
